自我介紹

澎澎老師好，我是家祥也可以叫我 Jack ，我是資訊工程學系畢業的，今年也要30歲了，在學期間其實就有接觸過 HTML 、 CSS 、 JavaScript ， 當時也覺得是個有趣的課程，畢業後一直都在家裡的蔬果行幫忙，前陣子因為疫情的關係，店裡的生意受到影響藉此機會我重拾了，以前在學校學的一些知識，用前端技術做了一個接單的表單網站，用 GAS 串接 google sheets 當作一個小型資料庫，在實作的過程中我覺得對於寫 code 還有完成一個應用這件事情，是一個很棒的感覺，之後也陸陸續續從網路上找到一些資源練習，切版RWD，以及串接一些 API 做練習，也下定決心想要出去外面找一份前端工程師的工作，剛好看到澎澎老師的這個訓練營，想讓自己有系統地好好學習一遍，希望能夠有這個機會！

問題回答

為什麼想成為前端、後端、或全端工程師？

想成為前端工程師是因為，本身興趣就是將事情簡單並有效率的處理，對於寫程式有一種熱情，遇到弄不清或者沒見過的東西 都會想尋求解答並解決，我覺得前端就是最接近使用者體驗的工作，而且技術繁多不同的公司需要不同的技術，也能成為學習 上面的動力，相對於後端工程師，我更喜歡看到畫面上的更動以及改變，從中可以獲得滿大的快樂。

為了成為軟體工程師，曾經做過什麼努力？

上網自己學習了一段時間的前端，透過 youtube 以及 udemy 看了許多相關的學習影片，澎澎老師的影片我在學生時期就有 看過，當時也是為了學習 HTML CSS ， 後來有看了 Amos 的頻道以及 Alex Jimmmy 的 JavaScript 教學分享，覺得 生在這網路發達的年代其實也不錯，許多資源都能查詢得到，也自己試著用學到的東西，解決身邊的問題，例如備忘錄 店裡的訂單管理，還有幫朋友做了一個簡單的一頁式購物。

如果參與這個計畫，會怎麼安排學習時間？

因為早上還是需要去店裡幫忙一下，所以中午結束後，會用 4 小時做練習，吃完晚餐後，用 2 ~ 3 小時整理練習遇到的困難，並找到方法解決。

如果參與這個計畫，預期會碰到什麼困難？你打算怎麼解決它？

我預期會碰到的困難是，如果進度跟不上或者因為時間上的壓力而導致體力沒辦法應付，不過這些都不是太大的問題，如果真的體力方面撐不住應該就先不會去店裡，先把學習的進度趕上了，有餘力了才去幫忙，這一次學習我希望是能夠將觀念還有熟悉程度都做一個大幅度的練習，如果有進度壓力，就花更多的時間去練習以及提問並且解決，不想要囫圇吞棗。

是否有想要加入的軟體公司？為什麼想加入該公司？

比較想加入的是 工作內容 偏電商開發或者系統開發，因為我個人對於電子商務的商業邏輯比較清楚，最近有看到寶雅總公司在找前端工程師，當時滿想去應徵，不過對於自己的信心還是不足，想要多練練有一個好一的的 side project 在嘗試應徵，而系統開發可以是類似 web app 之類的，對於應用層面的開發我更感興趣。

想要對我們說的事情？

感謝有這個訓練營的誕生，讓還在迷途不知道能不能夠轉職的我們有一條路可以選擇，雖然不知道能不能被選上，但是還是會繼續努力朝成為工程是了路繼續往前，也謝謝澎澎老師一直以來都是有閒暇就開直播，上傳影片分享一些開發上的內容。 希望你們能一直辦下去！ XD
